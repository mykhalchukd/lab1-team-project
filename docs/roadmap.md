Roadmap (План Розвитку)

Цей план охоплює основні функції, які будуть реалізовані протягом найближчих 3-4 тижнів, для переходу від базової системи до керованого каталогу.

Основні Етапи Розвитку

| Тиждень | Фокус | Завдання | Виконання |
| :--- | :--- | :--- | :--- |
| **3** | **Розширення API (Read)** | **Розширення API:** Додати новий endpoint `GET /items/{id}` для отримання детальної інформації про один товар за його унікальним ідентифікатором. | Бекенд (Node.js/Express) |
| **4** | **Безпека (Auth)** | **Базова Аутентифікація:** Створити таблицю `users` у БД. Реалізувати endpoint `POST /login` та `POST /register` для адміністраторів, використовуючи базове хешування паролів. | Бекенд/БД |
| **5** | **CRUD-операції (Admin)** | **Реалізація CRUD:** Додати захищені API-ендпоїнти (`POST`, `PUT`, `DELETE` на `/items`), які дозволять Адміністратору додавати, оновлювати та видаляти товари. | Бекенд |
| **6** | **UX та Пошук** | **Клієнтський Інтерфейс (UX):** Додати просте поле пошуку та базову логіку фільтрації на клієнтській стороні (`client/index.html`) для швидкого пошуку товарів за назвою. | Клієнт |

Примітка:

Робота над цим планом має відбуватися через постійний Git-цикл: створення гілки для кожної нової функції → Pull Request → Злиття в `main`.

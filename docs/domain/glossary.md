Ubiquitous Language (Скрізний глосарій)

Цей глосарій фіксує ключові терміни, які мають мати одне й те саме значення в усіх розмовах між бізнесом, розробниками та тестувальниками, у межах своїх контекстів.

Ключові терміни домену "Каталог товарів"

| Термін | Однозначне Визначення | Належність до Піддомену |
| :--- | :--- | :--- |
| **Товар (Item)** | Основна одиниця асортименту, що має унікальний ID, назву, ціну та опис. Це сутність, яка відображається в інтерфейсі. | Каталог |
| **Запас (Stock)** | Точна кількість одиниць певного **Товару**, наявних на складі, доступних для продажу. Облікова одиниця. | Запаси/Склад |
| **Адміністратор (Administrator)** | Аутентифікований **Користувач** із розширеною роллю, єдина особа, що має право виконувати операції CRUD (створення, редагування, видалення) даних. | Управління Користувачами |
| **Користувач (User)** | Будь-яка особа, яка переглядає **Каталог** (Visitor). У контексті Каталогу не має жодних прав, окрім читання. | Управління Користувачами / Каталог |
| **Категорія (Category)** | Сутність для логічного об'єднання **Товарів**. | Каталог |
| **Контекст (Bounded Context)** | Логічна межа, в якій терміни мають однозначне значення. | Архітектура |
